发布于 2020-08-11 / 更新于 2020-09-10 / Bangumi 有顶天家族 / 热度 0 / 喜欢 0

有顶天家族

我看番时总是全盘接受该作的设定，时而因为不假思索地接受设定而放弃思考，时而纠结于一些鸡毛蒜皮的剧情上的问题，这大概是傻瓜的血脉使然。ps: 人家总一郎只是把这当口头禅/奉为狸生信条，你们四兄弟是真あほう啊

「有顶天」- 来个港台式翻译我会很高兴的说

有頂天【うちょうてん】：忘乎所以，欣喜若狂。

网上查了一下，佛教用语“非想非非想天”的意思。日语中的俗意则是“得意忘形”的意思。
按照佛教用语来解释的话，到达这层境界的“人、神”已经是到达了最上层的境界，已经没有了“想”的念头，因此作“非想”，但并非没有所想，因此是“非非想”。
用剧中的话来形容就是：“世间万物有趣即可，（我们的）一切所行皆为愚昧的血之所然。”或者是：“好像无所想又好像有所想。”吧。大概指的就是主角一家。

有顶天、伊吕波……我个人不是很喜欢这种翻译啊，只有了解日本文化的观众才能反应过来这是啥。在我看来这种翻译唯一的好处是你看到就知道这是日本动画……嘛，这种对观众知识背景有要求的题目…我也不知道该说什么好。

要给动画第一季结局贴个标签的话，我会用“一出闹剧”

作者总是拿有趣出来说事，连总一郎最后对老二说的话也是有趣，

这世界……有些迷惑呢

这是一个天狗、人类、狸猫共存的世界，人类有对狸猫和天狗的认知。

所以，为什么三郎被拉去表演变身没人怀疑他是狸猫？弁天在屋顶上飞没人怀疑她是（伪）天狗？三郎那个姑且算他糊弄过去了，弁天那么明显在飞了却没暴露实在无法理解。

以人类的思维点评下鸭家四子（这里的人类是生活在现实世界中的我哟）

老大继承责任感
老二继承他悠哉的个性
老三继承他的傻劲
老幺继承他的纯真
（出自第8话21:20）

在我看来，老大继承了总一郎的责任感没错，但他没有能支撑这责任感的领导力，遇事慌乱无主见。他把维持下鸭家的荣光为己任，只是试图模仿伟大的爸爸走过的路，窃以为这是盲目愚昧的责任感。老大被这责任感束缚了个性，成了一个严肃刻板的老古董

老二，人形时放浪的发型，青蛙形态时弱受的语调，感觉有点不搭😂如果说他因内疚害死父亲而一直躲在井底逃避现实是悠哉，那作者还真是给悠哉两字抹黑。悠哉更像是作者给他套的帽子，老二在剧中的形象是一只看似佛系的青蛙，把“井底之蛙”、“可是我只是一只青蛙”挂在嘴边，讲些大道理，可是这不是他自卑恐惧无进取之下套的蛙皮面具（为了保护自己而产生的新的人格）吗。将自己完全置于下鸭一家之外，才能以完全局外人的视角观察事情的发展。

老三，男一号，倒是继承了他爸对“有趣”的追求。但这“有趣”是建立在自私自利之上的，爸爸为了有趣坐上老二变的电车把路上行人吓个半死，老三让放下身段求自己的老大下跪……他也有传统的地方（这还真是奇怪的反差设定），尊师，就像是师傅的保姆一样，这和有趣完全不搭边，如果这尊师是因为内疚于害师傅元气大伤，这就更不符合他一个利己主义者的设定了。至于“傻劲”我姑且理解为兼具“有趣”与“责任”的矛盾形象吧。

老幺他只是个孩子，不纯真才不正常啊喂

把这四个特性拆开来，放到这四子身上，我好像看到了四个作为人的反面教材。下鸭总一郎的伟大，是在于他能平衡融合这四种性格，而这四种性格单独拿出来都不足以使一个人名垂青史。

说句题外话：日漫中有太多神一样的父亲，明明孩子还没有展现出相应的才干，偏偏要把未来寄托在孩子身上（例：巨人，新妹）将情感置于理性之上，这种剧情着实有些反智，但偏偏有作者喜欢这种桥段。

弁天小姐真漂亮啊，不管是之前还是现在

作者还真是很会糊弄人呢

作品中的历史上的重大事件先由剧中人物讲话时用事件名轻描淡写地带过，然后扔下对事件一无所知一脸懵逼的观众，绝口不提事件的具体情况（每当这时我都会怀着“快来科普君给我来个痛快”的心情打开弹幕），在后面的故事中再讲述这个事件，ps: 事件名有很浓的日本味。这种故弄玄虚的讲述方式我很讨厌。

尤其是结局，尤其有被糊弄了的感觉。可能囿于主角组视角，动画未描述金曜日俱乐部6龙套眼中的结局（剩下的弁天姐姐和出现在主角的视角中所以不论，但），但习惯了以上帝视角看故事的我，不习惯这故事以这种不完全的方式收尾。金曜日俱乐部的人看到一屋子的狸猫还会计较教授抱走的一只吗？还有教授毫无预兆的180°思想转变：

教授之前一直强调“喜欢”和“吃”不矛盾，看到妈妈之后，却就算放弃金曜日俱乐部会员资格也要救妈妈（之前动画里还讲过不想退出俱乐部的说），说什么“吃是爱，舍不得吃也是爱”云云，诡辩大师的脑回路果然不是我能够理解的……
看到教授抱走妈妈那段，我立刻就想到了玛丽亚狂热里女主抢走与那国头上内衣的画面，感觉有点鬼畜。最后教授没有了那晚的记忆（怎么会？），却不为退出俱乐部感到遗憾，啊这……难道教授早就有退出之意，这次莫名其妙被踢了所以坦然接受？除了这个解释我想不通怎么解释以上现象。
俱乐部的人类被扇了后发生了什么，这年的尾牙宴这就么草草收场了吗？（毕竟这次狸猫要多少有多少，不吃可惜啊）

剧情漏洞警察（被大佬打脸倒计时开始）

夷川家的阿呆兄弟怎么知道三郎和教授很熟的？动画里总共见过两次啊（第5话和第9话）。如果阿呆知道教授和三郎很熟这条不成立，那么阿呆兄弟的阴谋成功的可能性微乎其微（除非三郎见大街上有人提笼子就去检查）；而如果阿呆兄弟知道很熟这条，说明阿呆兄弟的情报工作和监视能力不容小觑，这又和阿呆兄弟是阿呆这条相悖（除非扮猪吃老虎，与下鸭4笨蛋兄弟斗智斗勇）

这部作品有很多留白，如果有同人补完再好不过
