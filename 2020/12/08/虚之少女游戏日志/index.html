虚之少女游戏日志
发布于 2020-12-08 / 更新于 2020-12-11 / 日志 / 热度 0 / 喜欢 0

12/8

时隔6个月，终于重新开始推本作了，大概是从游戏内时间12/11开始。

推到游戏内时间12/19，真崎智之送惠梨回旅馆。

今天推的主要是日常。值得注意的只有真崎智之出院、跑腿、惠梨来事务所找人碰见真崎智之、真崎扔下工作跑去约会、真崎又扔下工作跑去约会。

然后关于“日本学生带便当到中午还是热的吗”这个问题，我和舍友讨论了下，然而涉及到物理知识，放弃。

12/9[^1]

推进到游戏内时间12/22，玲人准备去人形村。

跟着玲人的视角推进就是爽快。玲人见六识命，揭穿真崎智之真名毫不拖沓，验证了我昨天的推理（虽说我也在bangumi和别的渠道被剧透了点(bgm38)）。关于真崎智之的回忆部分，我本来期待会更精彩一些，至少也得和戌亥联手什么的，结果没有惊天动地的新发现，戛然而止了，有些遗憾。菜菜子嘛……当前时间线上的立绘眼神很奇怪，表情很奇怪，本来我还挺喜欢的说。接下来是圣诞，不过拖三天不会出什么幺蛾子吗？猜测下面进入剧情最精彩部分，毕竟前面铺垫有点太长了，希望能有值得我耐心看完这铺垫的后续发展。

还有，这作八木沼好像远比上一作讨人喜欢，有事直接找男主，就算损男主也不讨人厌（虽说这样我挺喜欢的啦，但是上一作八木沼挺讨人厌的人设就这么崩塌了）

玲人总喜欢开关于侵华的笑话，我还真是笑不出来。

[^1]: 码完这段文字的时间为12/10，16：32 am。

12/10[^2]

以为今天能有什么重大突破的，结果并没有，主角团依旧稳扎稳打（只因情报不足，没办法快速推进），冬史从秋五那搞来教团资料（秋五案子结束了还在追查，天助主角团，七七电话客串，纯粹是粉丝福利，只是不戳我的点）玲人则和戌亥会师人形集落。在集落，之前埋的几个伏笔（胶卷、丽人埋的匣子etc. ）被回收，激动人心（我都快忘了）。

主角团开始逐渐掌握事件相关的情报、玲人摸到了凶手杀人逻辑，相信玲人能很快把案子的各个碎片拼起来，还原真相。

主角进村的经历我觉得写的过于戏剧性了，进村时被当成外人受到敌视，但在黑矢医院掏出丈母娘千鹤的介绍信后，黑矢家态度立马大转变，黑矢家的爸爸和爷爷不仅表现出和善欢迎的态度，还聊起朽木家家常balabala。玲人与戌亥的接触也是，虽说见面时落了下风，失了话语权，但在新的死者出现后，玲人一转攻势，提出交易（结果玲人也没说什么对戌亥有帮助的事啊，几乎是空手套白狼），作为结果，得到远处观察现场权力，能把现场证物带回东京（这段是真的突兀，之前戌亥还一副不能让步的样子，突然就松口了），能把重要嫌疑人雏神兄妹带回东京，戌亥还倒贴了之前丽人都带不出驻在所的资料，大雪封路还派高山开车送到车站……这简直是亲儿子待遇啊！玲人觉得过意不去，就把丽人卖了（告诉戌亥关于丽人埋匣子的事，让戌亥代替自己去挖）。主角打算回东京那天，那个犀造老爷子（主角进村时用分量很重但勉强不失体面的话让玲人出村）已经开始跟玲人吐苦水了，仿佛玲人不是外人似的。

旅店居然收了张CG，是要点泡温泉泡晕的小老板娘浴巾七十几次，意外之喜。

今天的举证环节有点牵强，意义不大。好像是要选把东京事件与人形聚落事件联系起来的人，两个都选丽人（第一次选丽人，第二次选真崎，啊这……有什么意义吗）

推到这里，我想我可以来不负责任猜测下凶手是谁了。过去案子的凶手还不好猜，但现代案子的凶手应该是见过雏偶神的屈指可数的几个人之一：花恋、砂月（已死不可能）和雏神家的长辈。因为每次现场都有雏偶神，但是每次案发现场都进行了封锁，除了警察都不应该知道雏偶神泥人长什么样，连雏神家丽人都认不出雏偶神，所以凶手必定见过泥人，或间接知道泥人长什么样。菜菜子从不知哪儿见过雏偶神泥人，也有可能作案。再考虑犯案动机，我觉得花恋犯案的可能性最高。

推到游戏内时间12/29。
